track-record-buruk-endpoint
======================

Track Record Buruk Anggota DPR 2014

## How to setup

1. `$ bundle install`
2. Configure your database credentials in database.yml
3. `$ bundle exec rake db:create`
4. `$ bundle exec rake db:migrate`
5. `$ bundle exec rails server`
